    Linux-命令

    常用命令

    以下命令均需要安装后使用

    curl

    用于从命令进行数据传输，支持多种协议（HTTP、HTTPS、FTP等）。

    安装

    apt install curl

    GET请求

    curl https://www.example.com

    POST请求

    curl -d “param1=value1&param2=value2” https://www.example.com

    上传文件

    curl -F “file=@path/to/file”
    https://www.example.com/upload

    wget

    用于从网络下载文件，支持 HTTP、HTTPS 和 FTP 协议。

    安装

    apt install wget

    下载文件

    wget http://example.com/file.txt

    断点续传

    wget -c http://example.com/largefile.zip

    lynx

    基于文本的浏览器

    安装

    apt install lynx

    打开网页

    lynx https://www.example.com
    #箭头导航，回车确认

    nslookup

    查询DNS记录

    查询域名

    nslookup exampe.com

    指定DNS服务器

    nslookup example.com 8.8.8.8

    dig

    查询dns信息

    基本查询

    dig example.com

    查询特定记录类型

    dig example.com A

    ftp

    文件传输协议，用于在客户端和服务器之间传输文件

    链接FTP服务器

    ftp ftp.example.com

    上传文件

    put localfile.txt

    下载文件

    get remotefile.txt

    lftp

    功能强大的命令行文件传输工具，支持多种协议如 FTP、SFTP、HTTP 等

    连接到服务器

    lftp ftp://username:password@hostname

    上传文件

    put localfile.txt

    下载文件

    get remotefile.txt

    showmount

    显示 NFS 服务器的共享信息。

    显示共享目录

    showmount -e 192.168.1.1

    smbclient

    访问 SMB/CIFS 资源，类似于 FTP 客户端。

    连接到共享

    smbclient //server/share -U username

    上传文件

    put localfile.txt

    下载文件

    get remotefile.txt

    SSH

    安全登录、命令执行和数据传输。

    远程登录

    ssh username@hostname

    使用密钥登录

    ssh -i /path/to/private_key username@hostname

    telnet

    远程登录和网络测试。

    连接到服务器

    telnet 192.168.1.1 23

    traceroute

    跟踪网络中数据包的传输路径。

    跟踪路径

    traceroute example.com

    ntpdate

    手动同步计算机时间与远程 NTP 服务器时间。

    同步时间

    ntpdate time.ntp.org

    lsof

    列出系统中打开的文件‘

    查看所有打开的文件

    lsof

    查看特定进程的文件

    lsof -p

    namp

    网络发现和安全审计

    基本扫描

    nmap example.com

    扫描特定端口

    nmap -p 80 example.com

    mail

    发送和接受邮件

    发送邮件

    echo “邮件内容” | mail -s “主题” 收件人

    读取新邮件

    mail

    host

    查询DNS信息

    查询域名

    host example.com

    查询IP地址

    host 8.8.8.8
