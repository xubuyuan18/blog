华三交换机配置命令

华三-“交换机”

华三交换机命令

命令 功能说明
system-view 进入系统视图
interface GigabitEthernet0/0/0 进入以太网端口视图
display interface 接口流量、链路状态
vlan 2 **** 创建VLAN2
[H3C]undo vlan all 删除除缺省VLAN外的所有VLAN,缺省VLAN不能被删除
[H3C-vlan2]port G 0/4 to Ethernet0/7 将4到7号端口加入到VLAN2中
ip address192.168.1.2 255.255.255.0 为接口添加静态IP地址**
port link-type trunk 设置链路的类型为trunk
port access vlan 20 将当前access端口加入指定的VLAN
save 保存当前交换机配置
sys name 1 设置交换机名
